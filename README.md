# Поддерживается
стандартные ходы вперёд и назад (жульничать можно как и в обычной игре),
убийства одной пешки (нескольких убивают за n-ое колличество ходов).

# Не поддерживаются
временно не поддерживаются дамки,
отмена хода,
перезапуск игры.
